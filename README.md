**📊 Customer Behavior Analysis Project**
📌 **Overview**

This project focuses on analyzing customer shopping behavior to uncover patterns, identify key drivers of purchases, and provide actionable business insights.

The analysis follows an end-to-end data workflow:

Data preprocessing using Python
Business analysis using SQL
Data visualization using Power BI
Final insights presented through a report and a presentation

The goal is to help businesses improve **customer engagement, sales strategies, and decision-making.**

📂 **Dataset**
**Total Records:** ~3,900 transactions
**Features:** 18 columns
**Key Attributes:**
Customer details: Age, Gender, Location, Subscription Status
Purchase details: Item, Category, Amount, Season
Behavioral data: Discounts, Promo Codes, Purchase Frequency, Ratings, Shipping Type
**🛠️ Tools & Technologies**
**Python (Pandas, NumPy)**– Data cleaning & EDA
**SQL (SQL Server)** – Data analysis & querying
**Power BI** – Dashboard creation & visualization

**🔄 Project Workflow**

**1. Data Loading & Cleaning (Python)**
Loaded dataset using Pandas
Handled missing values (Review Ratings)
Standardized column names
Created new features (Age Groups, Purchase Frequency)

**2. Exploratory Data Analysis (EDA)**
Analyzed customer demographics
Identified spending patterns
Explored relationships between discounts, ratings, and purchases

**3. SQL Analysis**
Performed business-focused queries such as:
Revenue by customer segments
High-spending customers using discounts
Top-performing products
Customer segmentation (New, Returning, Loyal)
Shipping type impact on spending

**4. Data Visualization (Power BI)**
Built an interactive dashboard
Displayed KPIs like total customers, average purchase, and ratings
Visualized:
Revenue by category
Customer segments
Age group analysis
Subscription insights

**5. Reporting & Presentation**
Created a detailed project report
Designed a storytelling presentation (PPT) for stakeholders

**📊 Dashboard Highlights**
KPI Cards (Total Customers, Avg Purchase, Avg Rating)
Revenue & Sales by Category
Customer Segmentation
Subscription Analysis
Revenue by Age Group

**📈 Key Results & Insights**
Male customers contribute higher revenue
Express shipping users tend to spend more
Discounts increase sales but may impact profit margins
Loyal customers form the largest segment
Young adults generate the highest revenue

**💡 Business Recommendations**
Improve subscription benefits to increase adoption
Focus marketing on high-value customer segments
Optimize discount strategies to balance revenue and profit
Strengthen customer retention programs
Promote top-rated and high-performing products
